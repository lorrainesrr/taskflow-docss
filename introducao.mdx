# Introdução

A TaskFlow é uma API de gerenciamento de tarefas. Com ela, você cria projetos, organiza tarefas dentro desses projetos e acompanha o progresso de cada uma ao longo do tempo.

## O que dá para fazer

- Criar projetos para organizar seu trabalho
- Adicionar tarefas a esses projetos
- Listar todas as tarefas de um projeto
- Atualizar o status de uma tarefa conforme ela avança
- Apagar tarefas que não são mais necessárias

## Como a API funciona

A TaskFlow segue uma lógica de dependência entre os recursos: você precisa ter um projeto antes de criar tarefas, e precisa ter uma tarefa antes de atualizá-la ou apagá-la.

Essa ordem não é opcional. Se você tentar criar uma tarefa sem um `project_id` válido, a API vai retornar um erro.

O fluxo básico é sempre este:

```
Criar projeto → Criar tarefa → Atualizar ou apagar tarefa
```

## Antes de começar

Você vai precisar de uma chave de API para autenticar suas requisições. Veja como usá-la na página de [Autenticação](/autenticacao).
