# Criar tarefa

Cria uma nova tarefa dentro de um projeto existente. Toda tarefa é criada com o status `pending` por padrão.

**Você precisa de um `project_id` válido para usar este endpoint. Crie um projeto primeiro.**

---

## Requisição

```
POST /tasks
```

### Cabeçalhos

| Cabeçalho | Valor |
|---|---|
| `Authorization` | `Bearer tf-live-4a8f2c1e9b3d7e6a` |
| `Content-Type` | `application/json` |

### Body

| Campo | Tipo | Obrigatório | Descrição |
|---|---|---|---|
| `project_id` | string (uuid) | Sim | ID do projeto ao qual a tarefa pertence |
| `title` | string | Sim | Título da tarefa |
| `description` | string | Não | Descrição opcional da tarefa |

### Exemplo

```json
{
  "project_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "title": "Escrever página de introdução",
  "description": "Redigir visão geral e seção de primeiros passos."
}
```

---

## Respostas

### 201 — Tarefa criada

```json
{
  "task_id": "f9e8d7c6-b5a4-3210-fedc-ba9876543210",
  "project_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "title": "Escrever página de introdução",
  "description": "Redigir visão geral e seção de primeiros passos.",
  "status": "pending",
  "created_at": "2025-05-12T10:35:00.000Z",
  "updated_at": "2025-05-12T10:35:00.000Z"
}
```

| Campo | Tipo | Descrição |
|---|---|---|
| `task_id` | string (uuid) | Identificador único da tarefa. Use para atualizar ou apagar. |
| `project_id` | string (uuid) | ID do projeto ao qual a tarefa pertence |
| `title` | string | Título da tarefa |
| `description` | string ou null | Descrição da tarefa, se informada |
| `status` | string | Status atual. Sempre começa como `pending`. |
| `created_at` | string (date-time) | Data e hora de criação |
| `updated_at` | string (date-time) | Data e hora da última atualização |

### 400 — Requisição inválida

Ocorre quando `project_id` ou `title` estão ausentes.

```json
{
  "error": "Bad Request",
  "message": "The 'title' field is required and must be a non-empty string."
}
```

### 401 — Não autorizado

```json
{
  "error": "Unauthorized",
  "message": "Missing or invalid API key. Provide a valid key in the Authorization header as: Bearer <api_key>"
}
```

### 404 — Projeto não encontrado

Ocorre quando o `project_id` informado não existe.

```json
{
  "error": "Not Found",
  "message": "No project found with project_id 'abc123'. Create a project first using POST /projects."
}
```
