# Atualizar tarefa

Atualiza um ou mais campos de uma tarefa existente. Você pode alterar o status, o título ou a descrição. Pelo menos um campo precisa ser enviado.

---

## Requisição

```
PATCH /tasks/{task_id}
```

### Cabeçalhos

| Cabeçalho | Valor |
|---|---|
| `Authorization` | `Bearer tf-live-4a8f2c1e9b3d7e6a` |
| `Content-Type` | `application/json` |

### Parâmetros de URL

| Parâmetro | Tipo | Obrigatório | Descrição |
|---|---|---|---|
| `task_id` | string (uuid) | Sim | ID da tarefa que você quer atualizar |

### Body

Pelo menos um dos campos abaixo precisa ser enviado.

| Campo | Tipo | Valores aceitos | Descrição |
|---|---|---|---|
| `status` | string | `pending`, `in_progress`, `done` | Novo status da tarefa |
| `title` | string | — | Novo título da tarefa |
| `description` | string | — | Nova descrição da tarefa |

### Exemplo

```json
{
  "status": "in_progress"
}
```

---

## Respostas

### 200 — Tarefa atualizada

```json
{
  "task_id": "f9e8d7c6-b5a4-3210-fedc-ba9876543210",
  "project_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "title": "Escrever página de introdução",
  "description": "Redigir visão geral e seção de primeiros passos.",
  "status": "in_progress",
  "created_at": "2025-05-12T10:35:00.000Z",
  "updated_at": "2025-05-12T11:00:00.000Z"
}
```

O campo `updated_at` é atualizado automaticamente a cada alteração.

### 400 — Requisição inválida

Ocorre quando o valor de `status` não é um dos aceitos, ou quando nenhum campo é enviado.

```json
{
  "error": "Bad Request",
  "message": "Invalid status 'completed'. Allowed values are: pending, in_progress, done."
}
```

### 401 — Não autorizado

```json
{
  "error": "Unauthorized",
  "message": "Missing or invalid API key. Provide a valid key in the Authorization header as: Bearer <api_key>"
}
```

### 404 — Tarefa não encontrada

Ocorre quando o `task_id` informado não existe.

```json
{
  "error": "Not Found",
  "message": "No task found with task_id 'abc123'."
}
```
